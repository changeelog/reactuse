## Installation

```bash
npx changeelog/reactuse@latest add useTrigonometry
```

## Usage

```tsx
const {
  sin,
  cos,
  tan,
  asin,
  acos,
  atan,
  atan2,
  sinh,
  cosh,
  tanh,
  asinh,
  acosh,
  atanh,
} = useTrigonometry();

console.log(sin(Math.PI / 2)); // 1
console.log(cos(Math.PI)); // -1
console.log(tan(Math.PI / 4)); // ~1

try {
  console.log(asin(2)); // Throws an error
} catch (error) {
  console.error(error); // Error: asin calculation resulted in a non-finite number
}
```

## Reference

### useTrigonometry

`useTrigonometry` is a hook that provides validated trigonometric functions.
