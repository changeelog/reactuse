## Inslattion

```bash
npx changeelog/reactuse@latest add usePrecision
```

## Usage

```tsx
const value = 3.14159;
const precision = 2;
const formattedValue = usePrecision({ value, precision });

console.log(formattedValue); // 3.14
```

## Reference

### usePrecision

`usePrecision` is a hook that formats a numeric value to a specified precision.

<TypeTable
  type={{
    options: {
      description: 'Configuration object for the precision formatting.',
      type: 'PrecisionOptions',
    },

### PrecisionOptions

An interface defining the required configuration for precision formatting:

<TypeTable
  type={{
    precision: {
      description: 'The number of decimal places to round to.',
      type: 'Precision (number)',
    },
    value: {
      description: 'The numeric value to format.',
      type: 'NumericValue (number)',
    },
  }}
/>

- `precision` must be a non-negative integer.
- `value` must be a valid finite number.

### Return Value

Returns a `FormattedValue` (`number`) representing the input value rounded to the specified precision.

### Types

```ts
type Precision = number;
type NumericValue = number;
type FormattedValue = number;

interface PrecisionOptions {
  precision: Precision;
  value: NumericValue;
}
```
