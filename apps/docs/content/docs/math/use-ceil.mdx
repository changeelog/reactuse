title: useCeil

## Installation

```bash
npx changeelog/reactuse@latest add useCeil
```

## Usage

```tsx
import { useCeil } from '#';

function CeilingDisplay() {
  const value = 3.14;
  const ceiling = useCeil(value);

  return (
    <p>
      The ceiling of {value} is {ceiling}
    </p>
  );
}
```

## Reference

### useCeil

`useCeil` is a hook that calculates the ceiling of a given number or a function that returns a number.
