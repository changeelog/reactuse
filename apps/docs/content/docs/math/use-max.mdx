## Installation

```bash
npx changeelog/reactuse@latest add useMax
```

## Usage

```tsx
import { useMax } from '@changeelog/reactuse';

function MaxValueDisplay() {
  const maxValue = useMax(1, 5, 3, 9, 2);

  return <p>The maximum value is: {maxValue}</p>;
}
```

## Reference

### useMax

`useMax` is a hook that finds the maximum value among a set of numbers or functions that return numbers.
