## Installation

```bash
npx changeelog/reactuse@latest add useLogarithm
```

## Usage

```tsx
import { useLogarithm } from '#';

function LogarithmCalculator() {
  const { base, setBase, calculate } = useLogarithm({ initialBase: 10 });
  const value = 100;
  const result = calculate(value);

  return (
    <div>
      <p>
        log{base}({value}) = {result}
      </p>
      <button onClick={() => setBase(2)}>Change Base to 2</button>
    </div>
  );
}
```

## Reference

### useLogarithm

`useLogarithm` is a hook that provides functionality for logarithmic calculations with a customizable base.
