## Installation

```bash
npx changeelog/reactuse@latest add useInterpolation
```

## Usage

```tsx
import { useInterpolation, createPercentage } from '#';

function InterpolationExample() {
  const interpolate = useInterpolation();
  const start = 0;
  const end = 100;
  const amount = createPercentage(0.5); // 50%

  const result = interpolate(start, end, amount);

  return <p>Interpolated value: {result}</p>;
}
```

## Reference

### useInterpolation

`useInterpolation` is a hook that returns a function for linear interpolation between two numbers.

### Return Value

Returns an interpolation function with two possible signatures:
