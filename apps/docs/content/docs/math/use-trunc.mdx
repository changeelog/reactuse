Demo

## Installation

```bash
npx changeelog/reactuse@latest add useTrunc
```

## Usage

```tsx
const { value, truncate } = useTrunc(3.14);

console.log(value); // 3

truncate(5.67);
console.log(value); // 5

truncate(-7.89);
console.log(value); // -7
```

### Reference

### useTrunc

`useTrunc` is a hook that manages a truncated numeric value and provides a function to update it.
