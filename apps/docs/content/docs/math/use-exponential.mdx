## Installation

```bash
npx changeelog/reactuse@latest add useExponential
```

## Usage

```tsx
import { useExponential } from '#';

function ExponentialCalculator() {
  const { base, setBase, calculate } = useExponential({ initialBase: 2 });
  const exponent = 3;
  const result = calculate(exponent);

  return (
    <div>
      <p>
        {base}^{exponent} = {result}
      </p>
      <button onClick={() => setBase(3)}>Change Base to 3</button>
    </div>
  );
}
```

`useExponential` is a hook that provides functionality for exponential calculations with a customizable base.
