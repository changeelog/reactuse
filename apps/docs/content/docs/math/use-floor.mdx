## Installation

```bash
npx changeelog/reactuse@latest add useFloor
```

## Usage

```tsx
import { useFloor } from '#';

function FloorValueDisplay() {
  const value = 3.7;
  const floorValue = useFloor(value);

  return (
    <p>
      The floor value of {value} is {floorValue}
    </p>
  );
}
```

## Reference

### useFloor

`useFloor` is a hook that calculates the floor value of a given number or a function that returns a number.
