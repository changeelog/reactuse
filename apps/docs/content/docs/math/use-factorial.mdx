## Installation

```bash
npx changeelog/reactuse@latest add useFactorial
```

## Usage

```tsx
import { useFactorial } from '@changeelog/reactuse';

function FactorialCalculator() {
  const factorial = useFactorial();
  const number = 5;
  const result = factorial(number);

  return (
    <p>
      The factorial of {number} is {result.toString()}
    </p>
  );
}
```

## Reference

### useFactorial

`useFactorial` is a hook that returns a function for calculating factorials of non-negative integers.
