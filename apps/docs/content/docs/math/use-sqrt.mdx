## Installation

```bash
npx changeelog/reactuse@latest add useSqrt
```

## Usage

```tsx
const { value, setValue, sqrt } = useSqrt({ initialValue: 16 });

console.log(value); // 16
console.log(sqrt()); // 4

setValue(25);
console.log(sqrt()); // 5

setValue((prev) => prev + 11);
console.log(value); // 36
console.log(sqrt()); // 6
```
