## Installation

```bash
npx changeelog/reactuse@latest add useFibonacci
```

## Usage

```tsx
import { useFibonacci } from '@changeelog/reactuse';

function FibonacciSequence() {
  const { sequence, generate } = useFibonacci();

  const handleGenerate = () => {
    generate(10);
  };

  return (
    <div>
      <button onClick={handleGenerate}>Generate 10 Fibonacci Numbers</button>
      <p>Sequence: {sequence.join(', ')}</p>
    </div>
  );
}
```

## Reference

### useFibonacci

`useFibonacci` is a hook that provides functionality to generate Fibonacci sequences.
