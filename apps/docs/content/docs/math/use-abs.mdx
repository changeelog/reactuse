## Installation

```bash
npx changeelog/reactuse@latest add useAbs
```

## Usage

```tsx
import { useAbs } from '#';

function AbsoluteValueDisplay() {
  const value = -5;
  const absValue = useAbs(value);

  return (
    <p>
      The absolute value of {value} is {absValue}
    </p>
  );
}
```

## Reference

```typescript
export declare function useAbs(value: number | (() => number)): number;
```

## Reference

### useAbs

`useAbs` is a hook that calculates the absolute value of a given number or a function that returns a number.
