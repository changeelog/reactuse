## Installation

```bash
npx changeelog/reactuse@latest add usePow
```

## Usage

```tsx
const { base, setBase, exponent, setExponent, result } = usePow({
  initialBase: 2,
  initialExponent: 3,
});

console.log(base); // 2
console.log(exponent); // 3
console.log(result); // 8

setBase(3);
setExponent(2);
console.log(result); // 9
```

## Reference

### usePow

`usePow` is a hook that manages base and exponent values for power calculations, providing validation and real-time result computation.

### UsePowOptions

An interface defining optional configuration for the power calculation:

<TypeTable
  type={{
    initialBase: {
      description: 'The initial base value.',
      type: 'number',
      default: '0',
    },
    initialExponent: {
      description: 'The initial exponent value.',
      type: 'number',
      default: '1',
    },
  }}
/>

Both `initialBase` and `initialExponent` must be finite numbers.
