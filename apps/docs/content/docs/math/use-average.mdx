## Usage

```bash
import { useAverage } from '#';

function AverageDisplay() {
  const avg1 = useAverage(1, 2, 3, 4, 5);
  const avg2 = useAverage([10, 20, 30, 40, 50]);

  return (
    <div>
      <p>Average 1: {avg1}</p>
      <p>Average 2: {avg2}</p>
    </div>
  );
}
```

## Reference

### useAverage

`useAverage` is a hook that calculates the average of a set of numbers. It can accept either individual number arguments or an array of numbers.
