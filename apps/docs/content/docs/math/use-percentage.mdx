## Installation

```bash
npx changeelog/reactuse@latest add usePercentage
```

## Usage

```tsx
const { value, setPercentage, getAbsolute } = usePercentage({
  initialValue: 25,
  total: 200,
});

console.log(value); // 25
console.log(getAbsolute()); // 50
setPercentage(50);
console.log(getAbsolute()); // 100
```

## Reference

### usePercentage

`usePercentage` is a hook that manages a percentage value, providing validation and the ability to calculate the absolute value based on a total.

<TypeTable

### UsePercentageOptions

An interface defining optional configuration for the percentage:

<TypeTable
  type={{
    initialValue: {
      description: 'The initial percentage value.',
      type: 'number',
      default: '0',
    },
    total: {
      description: 'The total value to calculate the absolute value against.',
      type: 'number',
      default: '100',
    },
  }}
/>

The `initialValue` must be a finite number between 0 and 100, and `total` must be a finite number greater than 0.

### Return Value

Returns a UsePercentageReturn object with the following properties:

<TypeTable
