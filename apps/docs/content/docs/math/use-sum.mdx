## Installation

```bash
npx changeelog/reactuse@latest add useSum
```

## Usage

```tsx
const numbers = [1, 2, 3, 4, 5];
const sum = useSum(numbers);

console.log(sum); // 15
```

## Reference

### useSum

`useSum` is a hook that calculates the sum of an array of numbers.

### NumericArray

`NumericArray` is a type alias for a readonly array of numbers:

```ts
type NumericArray = readonly number[];
```

### Return Value

Returns a number representing the sum of all elements in the input array.
