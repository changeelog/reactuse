## Installation

```bash
npx changeelog/reactuse@latest add useTimeoutFn
```

## Usage

```tsx
const { isPending, start, stop } = useTimeoutFn(
  (message: string) => console.log(`Timeout: ${message}`),
  2000,
  { immediate: false },
);
```

## Reference

### useTimeoutFn

`useTimeoutFn` takes three parameters and returns a `UseTimeoutFnReturn` object.

An interface defining optional configuration for the timeout:
