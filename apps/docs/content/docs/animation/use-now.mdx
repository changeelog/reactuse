## Usage

```tsx
// Basic usage
const now = useNow();

// With controls
const { now, pause, resume, reset } = useNow({ controls: true });
```

## Reference

`useNow` is a hook that provides the current date and time, with optional controls for pausing, resuming, and resetting.

### Parameters

The hook accepts a single optional parameter:

<TypeTable
  type={{
    options: {
      description: 'Configuration object for the hook.',
      type: 'UseNowOptions',
      default: '{}',
    },
  }}
/>

### Options

The `options` object can include the following properties:

<TypeTable
  type={{
    controls: {
      description: 'Determines whether to return control functions.',
      type: 'boolean',
      default: 'false',
    },
    interval: {
      description:
        'The update interval in milliseconds or "requestAnimationFrame".',
      type: '"requestAnimationFrame" | number',
      default: '1000',
    },
    initialDate: {
      description: 'The initial date to use.',
      type: 'Date',
      default: 'new Date()',
    },
  }}
/>

### Return Value

The hook returns either a Date object or an object with the following properties, depending on the controls option:

<TypeTable
  type={{
    now: {
      description: 'The current date and time.',
      type: 'Date',
    },
    pause: {
      description: 'Function to pause the timer.',
      type: '() => void',
    },
    resume: {
      description: 'Function to resume the timer.',
      type: '() => void',
    },
    reset: {
      description: 'Function to reset the date.',
      type: '(date?: Date) => void',
    },
  }}
/>

## Notes

- If `controls` is set to `true`, the hook returns an object with the current date and control functions. Otherwise, it returns just the current date.
- The `interval` option can be set to a number (in milliseconds) or "requestAnimationFrame". If set to "requestAnimationFrame", the date will update on each animation frame.
- If a numeric `interval` is provided, it must be a positive number. Otherwise, an error will be thrown.
- The `reset` function allows you to set a new date. If no date is provided, it resets to the current date and time.
- The timer automatically starts when the hook is initialized. Use the `pause` and `resume` functions to control its execution.
