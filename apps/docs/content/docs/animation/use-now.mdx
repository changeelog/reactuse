## Installation

```bash
npx changeelog/reactuse@latest add useInterval
```

## Usage

```tsx
const { now, pause, resume } = useNow({ controls: true, interval: 1000 });
```

## Reference

### useNow

`useNow` is a custom hook that returns the current Date with optional controls.

<TypeTable
  type={{
    options: {
      description: '(Optional) Options for the useNow hook.',
      type: 'UseNowOptions & { controls?: T }',
      default: '{}',
    },
  }}
/>

### UseNowOptions

An interface defining the options for the useNow hook:

<TypeTable
  type={{
    controls: {
      description: 'Enable controls to pause and resume the interval.',
      type: 'boolean',
      default: 'false',
    },
    interval: {
      description:
        'The interval for the Date to update. Can be either a number in milliseconds or "requestAnimationFrame".',
      type: '"requestAnimationFrame" | number',
      default: '1000',
    },
  }}
/>

### Return Value

The return value of useNow depends on the controls option:

If `controls` is `true`, it returns an object of type UseNowReturnWithControls:

<TypeTable
  type={{
    now: {
      description: 'The current Date.',
      type: 'Date',
    },
    pause: {
      description: 'Function to pause the interval.',
      type: '() => void',
    },
    resume: {
      description: 'Function to resume the interval.',
      type: '() => void',
    },
  }}
/>

If `controls` is `false` or not provided, it returns the current Date.
