## Installation

```bash
npx changeelog/reactuse@latest add useIntervalFn
```

## Usage

```tsx
const { isActive, pause, resume } = useIntervalFn(
  () => console.log('Tick'),
  1000,
  { immediateCallback: true },
);
```

## Reference

`useIntervalFn` is a hook that executes a callback function at specified intervals and provides controls for managing the interval.

### Parameters

The hook accepts three parameters:

<TypeTable
  type={{
    callback: {
      description: 'The function to be executed at each interval.',
      type: '() => void',
    },
    interval: {
      description:
        'The interval duration in milliseconds or a function that returns the duration.',
      type: 'number | (() => number)',
    },
    options: {
      description: 'Optional configuration object.',
      type: 'UseIntervalFnOptions',
      default: '{}',
    },
  }}
/>

### Options

The `options` object can include the following properties:

<TypeTable
  type={{
    immediate: {
      description: 'Determines whether the interval starts immediately.',
      type: 'boolean',
      default: 'true',
    },
    immediateCallback: {
      description:
        'Determines whether the callback should be invoked immediately on start.',
      type: 'boolean',
      default: 'false',
    },
  }}
/>

### Return Value

The hook returns an object with the following properties:

<TypeTable
  type={{
    isActive: {
      description: 'Indicates whether the interval is currently active.',
      type: 'boolean',
    },
    pause: {
      description: 'Function to pause the interval.',
      type: '() => void',
    },
    resume: {
      description: 'Function to resume the interval.',
      type: '() => void',
    },
  }}
/>

## Notes

- The `callback` parameter must be a function. If it's not, the hook will throw a TypeError.
- The `interval` parameter must resolve to a positive number. If it doesn't, the hook will throw a RangeError.
- If an error occurs in the callback function, it will be caught and logged to the console, allowing the interval to continue running.
- The interval can be dynamically changed by passing a different value or function to the interval parameter.
- The `pause` and `resume` functions can be used to control the execution of the interval.
