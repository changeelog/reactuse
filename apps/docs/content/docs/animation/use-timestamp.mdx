## Installation

```bash
npx changeelog/reactuse@latest add useTimestamp
```

## Usage

```tsx
const { pause, resume } = useTimestamp({
  controls: true,
  offset: 0,
  immediate: true,
  interval: 1,
  callback: (ts) => {
    console.log('Timestamp:', ts);
  },
});
```

## Reference

### useTimestamp

The `useTimestamp` hook provides the current timestamp and optionally exposes controls for pausing and resuming updates.

### Return Value

If `controls` is `false`, the hook returns the current `timestamp` as a `number`.

If `controls` is `true`, the hook return an object containing the following properties:

  }}
/>
