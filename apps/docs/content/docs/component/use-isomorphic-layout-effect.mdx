## Installation

```bash
npx changeelog/reactuse@latest add useIsomorphicLayoutEffect
```

## Usage

```ts
useIsomorphicLayoutEffect(() => {
  // Your effect code here
}, [dependencies]);
```

## Reference

### useIsomorphicLayoutEffect

`useIsomorphicLayoutEffect` is a hook that provides a way to use `useLayoutEffect` in browser environments and falls back to `useEffect` in server-side rendering environments.

This hook does not return any value.
