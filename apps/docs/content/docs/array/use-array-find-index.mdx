## Installation

```bash
npx changeelog/reactuse@latest add useArrayFindIndex
```

## Usage 

```ts
const index = useArrayFindIndex(fruits, (fruit) => fruit.startsWith('a'));
```

## Reference

`useArrayFindIndex` is a hook that finds the index of the first element in an array that satisfies a given predicate function.

### Parameters
The hook accepts three parameters:

<TypeTable
type={{
list: {
description: 'The array to search or a function that returns an array.',
type: 'T[] | (() => T[])',
},
predicate: {
description: 'Function to test each element of the array.',
type: '(element: T, index: number, array: T[]) => boolean',
},
deps: {
description: 'Optional array of dependencies for the predicate function.',
type: 'React.DependencyList',
default: '[]',
},
}}
/>

### Return Value
The hook returns the index of the first element in the array that satisfies the predicate function, or -1 if no element is found.

<TypeTable
type={{
returnValue: {
description: 'The index of the found element or -1.',
type: 'number',
},
}}
/>

### Notes
If `list` is a function, it will be called to get the array.
If `list` is neither an array nor a function returning an array, an error will be thrown and -1 will be returned.
If the `predicate` is not a function, an error will be thrown and -1 will be returned.
