## Installation

```bash
npx changeelog/reactuse@latest add useArrayFind
```

## Usage

```ts
const foundElement = useArrayFind(array, (element) => element > 5);
```

## Reference

`useArrayFind` is a hook that finds the first element in an array that satisfies a given predicate function.

### Parameters

The hook accepts three parameters:

<TypeTable
  type={{
    list: {
      description: 'The array to search or a function that returns an array.',
      type: 'T[] | (() => T[])',
    },
    predicate: {
      description: 'Function to test each element of the array.',
      type: '(element: T, index: number, array: readonly T[]) => boolean',
    },
    deps: {
      description: 'Optional array of dependencies for the predicate function.',
      type: 'readonly unknown[]',
      default: '[]',
    },
  }}
/>

### Return Value

The hook returns the first element in the array that satisfies the predicate function, or undefined if no element is found.

<TypeTable
  type={{
    returnValue: {
      description: 'The found element or undefined.',
      type: 'T | undefined',
    },
  }}
/>

## Notes

- If `list` is a function, it will be called to get the array.
- If `list` is neither an array nor a function returning an array, a `TypeError` will be thrown.
