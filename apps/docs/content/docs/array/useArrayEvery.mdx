## Installation

```bash
npx changeelog/reactuse@latest add useArrayEvery
```

## Usage

```ts
const allPositive = useArrayEvery([1, 2, 3, 4, 5], (num) => num > 0);
console.log(allPositive); // true

const allEven = useArrayEvery([2, 4, 6, 7, 8], (num) => num % 2 === 0);
console.log(allEven); // false
```

## Reference

`useArrayEvery` is a hook that checks if all elements in an array satisfy a given predicate function.

### Parameters

The hook accepts two parameters:

### Return Value

The hook returns a boolean value:

<TypeTable
  type={{
    result: {
      description:
        'True if all elements satisfy the predicate, false otherwise.',
      type: 'boolean',
    },
  }}
/>

## Notes

- If the `list` parameter is `null`, `undefined`, or an empty array, the hook returns `true`.
- The predicate function receives three arguments: the current element, its index, and the array itself.
