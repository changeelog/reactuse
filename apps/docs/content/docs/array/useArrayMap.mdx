## Installation

```bash
npx changeelog/reactuse@latest add useArrayMap
```

## Usage

```ts
const numbers = [1, 2, 3, 4, 5];
const doubledNumbers = useArrayMap(numbers, (num) => num * 2);
```

## Reference

`useArrayMap` is a hook that applies a given function to each element of an array and creates a new array with the results.

### Parameters

The hook accepts two parameters:

<TypeTable
  type={{
    list: {
      description: 'The array to map or a function that returns an array.',
      type: 'MaybeRef<readonly MaybeRef[]>',
    },
    fn: {
      description: 'The function to apply to each element of the array.',
      type: '(element: T, index: number, array: readonly T[]) => U',
    },
  }}
/>

### Return Value

The hook returns a new array with the results of calling the provided function on every element in the input array.

<TypeTable
  type={{
    returnValue: {
      description: 'The new mapped array.',
      type: 'readonly U[]',
    },
  }}
/>

## Notes

- The `list` parameter can be either a direct array or a function that returns an array (`MaybeRef` type).
- Each element in the `list` can also be a `MaybeRef`, allowing for dynamic values.
- The resulting array is frozen (`Object.freeze`) to prevent accidental mutations.
