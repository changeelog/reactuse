## Installation

```bash
npx changeelog/reactuse@latest add useArrayReduce
```

## Usage

```ts
const numbers = [1, 2, 3, 4, 5];
const sum = useArrayReduce(numbers, (acc, curr) => acc + curr, 0);
```

## Reference

`useArrayReduce` is a hook that applies a reducer function to an array, reducing it to a single value.

### Parameters

The hook accepts three parameters:

<TypeTable
  type={{
    list: {

    },
    reducer: {
      description: 'The reducer function to apply to the array.',

    },
    initialValue: {
      description: 'The initial value for the reduction.',
      type: 'U',
    },
  }}
/>

### Reducer Function

The reducer function should have the following signature:

<TypeTable
  type={{

    currentIndex: {
      description: 'The index of the current element being processed.',
      type: 'number',

    array: {
      description: 'The array being reduced.',
      type: 'readonly T[]',
    },
  }}
/>

### Return Value

The hook returns the final reduced value.

<TypeTable
  type={{
    returnValue: {
      description: 'The result of reducing the array.',
      type: 'U',
    },
  }}
/>

## Notes

- If the `list` is `null` or `undefined`, a warning is logged and the initialValue is returned.
- If an error occurs during the reduction process, it's caught, logged to the console, and the `initialValue` is returned.
- If `list` is not an array, or if `reducer` is not a function, a `TypeError` will be thrown.
