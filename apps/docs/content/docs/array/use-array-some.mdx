## Installation

```bash
npx changeelog/reactuse@latest add useArraySome
```

## Usage

```ts
const numbers = [1, 2, 3, 4, 5];
const hasEvenNumber = useArraySome(numbers, (num) => num % 2 === 0);
```

## Reference

`useArraySome` is a hook that tests whether at least one element in the array passes the test implemented by the provided predicate function.

### Parameters

The hook accepts two parameters:

<TypeTable
  type={{
    list: {
      description: 'The array to check or a function that returns an array.',

    },
    predicate: {
      description: 'Function to test each element of the array.',
      type: '(element: T, index: number, array: readonly T[]) => boolean',
    },
  }}
/>

### Return Value

The hook returns a boolean indicating whether at least one element in the array satisfies the provided testing function.

<TypeTable
  type={{
    returnValue: {

      type: 'boolean',
    },
  }}
/>

## Notes

The list parameter can be either a direct array or a function that returns an array (`MaybeRef` type).
Each element in the `list` can also be a `MaybeRef`, allowing for dynamic values.
If an error occurs during the execution (e.g., invalid input), it's caught, logged to the console, and `false` is returned.
