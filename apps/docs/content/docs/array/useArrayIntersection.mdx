## Installation

```bash
npx changeelog/reactuse@latest add useArrayIntersection
```

## Usage

```ts
const intersection = useArrayIntersection(
  () => [1, 2, 3],
  () => [2, 3, 4],
  () => [3, 4, 5],
);
console.log(intersection); // [3]
```

## Reference

`useArrayIntersection` is a hook that returns the intersection of multiple arrays.

### Parameters

The hook accepts a variable number of parameters:

<TypeTable
  type={{
    arrays: {
      description:
        'Arrays or functions that return arrays to find the intersection of. At least one array must be provided.',
      type: 'ArrayOrGetter[]',
    },
  }}
/>

### Return Value

The hook returns an array containing the intersection of all input arrays:

<TypeTable
  type={{
    intersection: {
      description: 'An array of elements that are common to all input arrays.',
      type: 'T[]',
    },
  }}
/>

## Notes

- If an array is provided as a function, it will be called to retrieve the array value.
- If the input arrays are empty, the hook will return an empty array.
- If only one array is provided, the hook will return that array.
