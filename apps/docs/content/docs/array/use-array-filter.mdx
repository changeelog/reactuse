## Installation

```bash
npx changeelog/reactuse@latest add useArrayFilter
```

## Usage

```ts
const numbers = [1, 2, 3, 4, 5];
const evenNumbers = useArrayFilter(numbers, (num) => num % 2 === 0);
```

## Reference

`useArrayFilter` is a hook that filters an array based on a predicate function and returns the filtered array.

### Parameters
The hook accepts two parameters:

The predicate function has the following signature:

array: {

}}
/>

### Return Value
The hook returns a filtered array of the same type as the input:

<TypeTable
type={{
filteredArray: {
description: 'The resulting array after applying the filter.',
type: 'readonly T[]',
},
}}
/>

## Notes
- If the input is a function, it will be called to get the array.
- If an error occurs during filtering, it will be caught, logged to the console, and an empty array will be returned.

The filtered array is readonly to prevent accidental mutations. If you need to modify the array, create a copy first.
